Lesson 11: Search Algorithms and Trace Tables

Apply linear and binary search and trace each step clearly.

Topics: linear search / binary search / trace table / dry run

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Exam prep starts with search logic and accurate trace tables.

Key Points

  • Linear search works on unsorted lists.
  • Binary search requires sorted data.
  • Trace tables must show variable changes each step.
  • Explain why an algorithm stops when a match is found.

Example: linear search trace

target = 9
values = [4, 7, 9, 12]
for i in range(len(values)):
    if values[i] == target:
        print(i)

Mini Practice
