Lesson 9: Mixed Problem Solving

Combine selection, loops, and data structures in one task.

Topics: decomposition / test cases / algorithm steps

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

This lesson combines core skills into small algorithmic problems.

Key Points

  • Break a task into clear steps before coding.
  • Write quick tests for normal and edge cases.
  • Use loops, conditionals, and structures together.
  • Refactor repeated logic into helper functions.

Example: average with pass/fail

scores = [70, 80, 90]
avg = sum(scores) / len(scores)
print('Pass' if avg >= 60 else 'Retry')

Mini Practice

  • Compute average of four numbers.
  • Print 'even' or 'odd' for each number in a list.
