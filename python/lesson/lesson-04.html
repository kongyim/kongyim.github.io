Lesson 4: Strings and Text Processing

Manipulate text using indexing and string methods.

Topics: len / split / strip / indexing

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

String methods help you clean and split real user input.

Key Points

  • len(text) gives character count.
  • text.strip() removes edge spaces.
  • text.split(',') turns CSV-like text into a list.
  • Indexing starts at 0.

Example: parse comma input

raw = 'red,blue,green'
parts = raw.split(',')
print(parts[0])
