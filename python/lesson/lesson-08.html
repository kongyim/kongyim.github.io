Lesson 8: Files and JSON Data

Work with text files and JSON data structures.

Topics: open / read/write modes / json / data files

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Files and JSON are used to store and load structured data.

Key Points

  • Use 'r' to read and 'w' to write files.
  • json.dumps converts Python object to JSON string.
  • json.loads parses JSON string to Python object.
  • with open(...) closes files safely.

Example: write JSON text

import json
record = {'name': 'Ava', 'score': 90}
json_text = json.dumps(record)
