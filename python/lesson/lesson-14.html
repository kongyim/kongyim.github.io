Lesson 14: Validation, Testing, and Robust Programs

Design stronger programs with validation and test planning.

Topics: validation / test data / boundary values / defensive coding

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Robust programs handle invalid input and include a test plan.

Key Points

  • Validation checks format, range, and type.
  • Use normal, boundary, and invalid test data.
  • Boundary tests target edge values.
  • while loops can enforce repeated input until valid.

Example: mark validation loop

valid = False
while not valid:
    mark = int(input('Mark: '))
    valid = 0 <= mark <= 100

Mini Practice

  • List 2 valid and 2 invalid marks for range 0..100.
  • Add try/except to catch non-numeric input.
