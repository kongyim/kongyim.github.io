Lesson 3: Iteration with for and while

Use loops to repeat actions and build totals.

Topics: for / while / range / accumulator

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Loops let you repeat logic reliably and avoid copy-paste coding.

Key Points

  • for loops are great for known collections/ranges.
  • while loops repeat while a condition remains true.
  • range(start, stop) excludes stop.
  • Use accumulators like total += value.

Example: sum first five numbers

total = 0
for i in range(1, 6):
    total += i
print(total)
