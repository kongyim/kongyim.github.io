Lesson 12: Sorting Algorithms (GCSE Core)

Understand and trace simple sorting, especially bubble sort.

Topics: bubble sort / swap / pass / algorithm steps

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Sorting questions often assess algorithm steps and swapping logic.

Key Points

  • Bubble sort compares adjacent values.
  • A swap happens when pair order is wrong.
  • If no swaps happen in a pass, list is sorted.
  • Use clear loop boundaries to avoid index errors.

Example: one bubble pass

items = [5, 2, 4]
for i in range(len(items) - 1):
    if items[i] > items[i + 1]:
        items[i], items[i + 1] = items[i + 1], items[i]
