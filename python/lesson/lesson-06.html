Lesson 6: Functions and Decomposition

Write reusable functions with parameters and return values.

Topics: def / parameters / return / modularity

Quiz timer: 20 minutes (auto-submit on timeout)

Lesson Notes

Functions help you break problems into reusable parts.

Key Points

  • Define with def name(parameters):
  • Use return to send a result back.
  • Good function names describe behavior.
  • Smaller functions are easier to test.

Example: reusable grading

def grade(score):
    if score >= 60:
        return 'Pass'
    return 'Retry'
